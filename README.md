# 8th Wonder — Drone Services Website

## Project Overview

A professional marketing website for **8th Wonder**, a drone photography and videography business based in **South East Queensland, Australia**. The site is designed to showcase the operator's work and convert visitors into leads across four core industries.

---

## Business Context

- **Business name:** 8th Wonder
- **Location:** South East Queensland, QLD, Australia
- **Operator:** Professional licensed drone operator
- **Existing clients:** Construction companies, real estate agents, event organisers, agricultural businesses

---

## Website Objectives

1. **Showcase the portfolio** — Display high-quality aerial photography and videography work to impress potential clients
2. **Generate leads** — Drive enquiries and quote requests from new clients across all four industries
3. **Build credibility** — Establish trust through testimonials, an about section, and professional presentation
4. **Expand the business** — Help the operator reach new clients beyond existing word-of-mouth referrals

---

## Target Audiences

| Industry | What they need |
|---|---|
| Construction companies | Site progress photography, surveying, inspections |
| Real estate agents | Property aerial photography and video walkthroughs |
| Event organisers | Aerial event coverage, highlight reels |
| Agriculture | Crop monitoring, land mapping, farm surveys |

---

## Tech Stack

| Tool | Purpose |
|---|---|
| React (Vite) | UI framework and build tool |
| Tailwind CSS v4 | Utility-first styling |
| Lucide React | Icon library |

---

## Site Structure

1. **Navigation** — Fixed top nav with logo and anchor links + mobile menu
2. **Hero** — Full-screen cinematic intro with headline and CTA
3. **Services** — Four service cards (Construction, Real Estate, Events, Agriculture)
4. **Portfolio** — Filterable image gallery (All / Construction / Real Estate / Events / Agriculture)
5. **About** — Operator background, experience, equipment, certifications
6. **Testimonials** — Client reviews and social proof
7. **Contact** — Quote request form and contact details
8. **Footer** — Links, social media, location

---

## Design Direction

- **Style:** Clean and professional (light, minimal)
- **Tone:** Premium, trustworthy, cinematic
- **Colour palette:** Dark navy (#0f172a) / White / Gold accent (#f59e0b)
- **Typography:** Inter / system sans-serif

---

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
